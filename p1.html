AJAX

Asynchronous Javascript and XML, or Ajax , is not a technology in intself,but rather an approach to using a number of existing technologies together ,including HTML or XHTML,CSS, JAVASCRIPT,DOM,XML,XSLT, and most importantly the XMLHttpRequest object.

External js function execution using call back (Asynchronous)

Asynchronous code allows the program to be executed imediatly.

Asynchronous programming allows you to perform multiple requests simultaneously and complete more tasks faster.
